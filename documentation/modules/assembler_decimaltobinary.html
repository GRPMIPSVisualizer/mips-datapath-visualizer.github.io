decimalToBinary

  • decimalToBinary(decimal: number, numOfBits: number): string
    • Defined in Assembler/DecimalToBinary.ts:15

    Transform a number from decimal to binary. The input numOfBits must be sufficient to represent the input decimal number.

    Parameters

    • decimal: number

      the decimal number that need translation.

    • numOfBits: number

      the number of bits of the desired binary number (or two's implement). If the input decimal number is positive, it will be translated into its binary number or its two's complement according to numOfBits. If the input decimal number is negative, it will be translated into its two's complement.

    Returns string

    a string of the binary format of the decimal number.
