Class DFlipFlop

Class DFlipFlop is an abstract model of real DFlipFlap This is a down-edge trigger DFlipFlop,which means the data in D signal will only stored into this DFlipFlop when clock signal is changed from high to low
DFlipFlop is the basic unit of 32bits register.
1 DFlipFlop can store 1 bit data
