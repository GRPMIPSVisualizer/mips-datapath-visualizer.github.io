Class singleCycleCpu

This is the real CPU of MIPS
Assembler and all circuit's components are integrated in this class
The functionalities of this class simulates a real MIPS CPU.
The pipeline is not realized is this class so we gave it name singleCycleCPU which means the cpu will only run one instruction in one cycle.
