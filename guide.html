Navigation bar function

1. Home
The first button in the navigation bar is the Home button, which takes you quickly back to the home page.
2. Help
The second button in the navigation bar is the Help button, which is clicked to take you to the Help page. See Help Page section for more details.
3. Doc
The third button in the navigation bar is the Doc button, clicking it will take you to the Documentation page. See Documentation Page section for more details.
4. Test
The fourth button in the navigation bar is the Doc button, clicking it will take you to the Documentation page. See section 7 Documentation Page for more details.
5. Demo
The fifth button in the navigation bar is the Demo button. After you click this button, a drop-down menu will appear. You can choose one of the demos you want to see and click. The code input area on the right side of the home page will automatically display the corresponding MIPS code.
Code input area function
1. Upload
The first button in the code entry area is the upload button. When you click this button, a file selection box pops up and you can select files with the suffix .s or.asm to upload. When you click OK, the contents of the selected file are displayed in the code input area below the button.
2. Download
The second button in the code entry area is the download button. After you click the button, the code in the code input area is saved in a file called demo.asm and downloaded locally.
3. Compile
The third button in the code entry area is the compile button. When you have entered the code, you can compile it by clicking this button. If compiled successfully, the data is automatically saved in the appropriate location; If the compilation was unsuccessful, it will be notified in the display area at the bottom of the page. This feature is disabled while the animation is running.
Animation function
1. Step forward
The first button in the animation section is the step forward button. After clicking this button once, the animation will automatically play forward a step. This feature is disabled when code is not compiled.
2. Play
The second button in the animation section is the play button. After clicking this button, the animation of all commands will automatically play continuously. This feature is disabled when code is not compiled.
3. Pause
The third button in the animation section is the pause button. Click this to pause the animation while you're playing the animation. Click again to resume the animation. This feature is disabled when code is not compiled.
4. Terminate
The fourth button in the animation section is the terminate button. While the animation is running, no other actions can be taken at will. So when you want to stop the entire animation run and do something else, you can stop the animation process by clicking this button and every button will be undisable.
5. Speed changing
The last 2 buttons are the buttons to change the animation speed. The left button can reduce the speed by half, and the right one can double the current speed.
